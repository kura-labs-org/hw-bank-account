# hw-bank-account

To run the program type
`python BankSession.py`

There are three accounts you can use as soon as the session loads.

| firstname | lastName | password | balance |
| ----  | ------ | ------ | ----- |
| Isaac | Barrezueta | password1 | 1000 | 
Dan | C | password2 | 0
Mo | D | password3 | 0

## Actions you can do

* check balance
* deposite money
* withdraw money
* transfer funds
* get transaction history
* exit
